## Development

    nvm install
    nvm use
    npm i

### Lint

    npm run lint


### Tests

Run tests

    npm test

Coverage

    # without reporter
    npm run cov
    
    # See coverage in browser
    npm run cov-html

    # With full reporter
    npm run coverage
    

### Documentation

Generate documentation

    # Generate and open in browser
    npm run doc-html
    
    # Just generate
    npm run doc
